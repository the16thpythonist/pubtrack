Welcome to pubtrack's documentation!
====================================

*pubtrack* is an application for tracking scientific publications of a set of authors from various secondary data
locations, currently supporting *Scopus* and *KITOpen*. Pubtrack has been developed for the Institute for Data
Processing and Electronics (IPE) at the Karlsruhe Institute of Technology (KIT) as an administrative tool to keep
track of changes and potential issues in publication data.
